## Velatio


Låter som en latinsk böjning av velatio – faktiskt ett riktigt latinskt ord
Betyder "täckandet, slöjandet, döljandet" – en handling, inte bara ett tillstånd
Användes i romerska ritualer när man täckte huvudet som tecken på respekt/doldhet
Ingen negativ koppling
Låter som ett tech-bolag eller AI-produkt direkt
